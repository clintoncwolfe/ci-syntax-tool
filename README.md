# CI Syntax Tools

## Overview

Checks various source file types for syntax errors, and formats the results in a flexible way.

## Syntax Plugins

So far, we support:

None.

### Adding Your Own

Extend CI::Syntax::Language, and use --require to include your library.

## Output Plugins

So far, we support:

None.
